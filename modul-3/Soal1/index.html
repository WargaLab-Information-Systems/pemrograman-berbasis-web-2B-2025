Tentang Saya

Saya adalah mahasiswa aktif Program Studi Sistem Informasi di Universitas Trunojoyo Madura. Saya memiliki ketertarikan yang besar dalam bidang teknologi dan selalu berusaha untuk terus berkembang dengan mempelajari berbagai hal baru di dunia IT.

Saya berkomitmen untuk menerapkan pengetahuan yang saya peroleh di bangku kuliah untuk menciptakan solusi yang bermanfaat, terutama dalam pengembangan sistem informasi dan aplikasi berbasis web. Saya juga aktif mengembangkan keterampilan di berbagai bahasa pemrograman, seperti HTML, CSS, JavaScript, dan Python.

Selain itu, saya memiliki minat dalam pengembangan web dan aplikasi mobile. Saya percaya bahwa dunia teknologi memiliki potensi besar untuk memberikan dampak positif bagi masyarakat, dan saya ingin menjadi bagian dari perubahan itu.
